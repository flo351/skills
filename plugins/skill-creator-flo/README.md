# skill-creator-flo

> Coordinate creation or modification of Claude Code skills — personal marketplace (Type A, stable or beta), standalone (Type B), or other tools (Type C). Bootstraps personal GitHub marketplaces if missing. Invoke when the user wants to create, scaffold, or edit a skill.

- **Created**: `2026-05-20`
- **Last updated**: `2026-05-20`

```mermaid
flowchart LR
    A[user request] --> B[skill-creator-flo]
    B --> C[skill files]
```

## Installation

```
/plugin marketplace add flo351/skills
/plugin install skill-creator-flo@skills
```

Slash: `/skill-creator-flo`. Update: `/plugin marketplace update skills`.
